# SAT1600.uz Private Pilot Checklist

Run this before any public launch.

## Group

- 5-10 real students
- Mixed ability levels
- Use one diagnostic mock only

## Observe

- Where students pause for a long time
- Where they change answers
- Where they guess quickly
- Questions that feel weird, unfair, or unlike SAT wording
- Distractors that nobody chooses
- Distractors that too many strong students choose

## Admin Review

After each pilot attempt, open the admin quality dashboard and review:

- high hesitation
- high answer-change rate
- weak distractor effectiveness
- high drop-off
- unexpected low percent-correct
- flagged student comments

Use statuses:

- `approved`: item is ready
- `reviewed`: checked but still needs more data
- `flagged`: wording/content needs human review
- `disabled`: do not show this item

Keep recalibration frozen until each question has at least 200 real attempts.
